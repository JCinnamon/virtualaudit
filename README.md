# urban-change-observer — Urban Change Observer

A browser-based tool for conducting systematic visual audits of urban change.
Supports structured data collection on urban features using multitemporal street view imagery.

**[Live →](https://www.jonathancinnamon.com/urban-change-observer/)**

---

## Features

- Integrated google street view panoramic viewer
- Structured audit form with fields for observer ID, imagery date, feature type, change type, and estimated year of change. Download data with location (lat/lon coordinates)
- Address search to navigate to any location
- Designed for systematic data collection across multiple sites

## Usage

Open the tool in a browser. Search for an address or navigate the map to your target location. Complete the audit form fields for each site. Download as CSV file. No account or installation required.
